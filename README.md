# Trading Agent

An AI-powered trading agent built with LLMs.

## Overview

This project implements an autonomous trading agent that leverages large language models to analyze market data, generate insights, and make trading decisions.

## Features

- Market data analysis
- LLM-driven decision making
- Automated trade execution

## Getting Started

### Prerequisites

- Python 3.10+

### Installation

```bash
git clone <repo-url>
cd trading_agent
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
